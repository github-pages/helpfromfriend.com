How to install Deadbeef in Ubuntu

0

If you don't know DeaDBeeF yet I highly recommend to check it out - it's lightweight music player for linux, with simple and easy to work with user interface. It MS Windows alternative would be foobar which has pretty the same level of awesomeness as DeaDBeeF.. Unfortunately there is no official package for new Ubuntu 12.10 Quantal. But thanks to alternative user package repositories you can very easy install the player through standardÂ  Ubuntu way.

$ sudo add-apt-repository ppa:alex-p/notesalexp-quantal$ sudo apt-get update$ sudo apt-get install deadbeef

Enjoy!
